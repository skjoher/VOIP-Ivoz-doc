.. _brand_services:

################
Generic Services
################

This section allows the brand operator to change the default services and default service codes for new clients.

By default this list has all the services and codes from the god level **Service** section.

.. warning:: Changing the default code in this section will only affect new created clients. Existing clients codes won't
          be modified.

.. warning:: Deleting a service will delete this service for all existing clients.
